# Créer un projet dans PyCharm

Un **projet PyCharm** est un dossier contenant tous les fichiers liés à un programme Python.
Il contient habituellement :

- des fichiers `.py` (code Python)
- un environnement virtuel (`venv`) pour gérer les dépendances
- d'autres fichiers comme des notebooks Jupyter, des images, ou des fichiers Markdown

---

## Étapes pour créer un projet

1. Ouvrir **PyCharm**.
2. Sur l’écran d’accueil, cliquer sur **New Project** (ou **File > New Project**
si un autre projet est déjà ouvert).
3. Choisir un **emplacement** pour le projet, idéalement dans votre [repo GitHub](creer-repo-github)
(par exemple : `Desktop/mon_repo/nom_du_nouveau_projet`).
4. Choisir un **environnement Python** :
   - Cocher *Create a welcome script* pour qu'un fichier `main.py` soit créé automatiquement.
   - Laisser l’option *Project venv* cochée pour créer un environnement isolé.
   - Vérifier que l’interpréteur sélectionné est bien une version de **Python 3.x**.
5. Cliquer sur **Create**.
6. Si vous avez déjà un projet ouvert, cliquer sur **New Window**
pour ouvrir le nouveau projet dans un nouvel onglet.

---

## Comprendre la structure du projet

Une fois le projet créé, PyCharm affiche :

- un dossier principal (nom du projet)
- un dossier `venv/` contenant l’environnement virtuel
- un fichier `main.py` (optionnel)
- l’éditeur principal au centre, l’arborescence des fichiers à gauche

---

## Ajouter un premier fichier

1. Dans la colonne de gauche, cliquer-droit sur le dossier du projet.
2. Choisir **New > Python File**.
3. Donner un nom au fichier (ex : `test`) et appuyer sur ⏎.
4. Un nouveau fichier `test.py` est créé.

---

## Astuces

- Toujours créer un projet avant d’écrire du code, pour bien organiser vos fichiers.
- Éviter les accents, espaces ou caractères spéciaux dans les noms de dossiers.
- Le projet peut contenir aussi des notebooks Jupyter, des fichiers Markdown, etc.
