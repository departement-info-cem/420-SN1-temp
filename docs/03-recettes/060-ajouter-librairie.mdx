# Ajouter une librairie

La plupart des librairies Python sont disponibles sur le site [PyPi](https://pypi.org/).
Une *librairie* (ou *package*) contient du code Python qu'on peut réutiliser dans nos projets pour éviter
de tout programmer soi-même.

---

## Ajouter une librairie à votre projet

1. Ouvrir le projet dans PyCharm.
2. Allez dans **File > Settings**.
3. Dans la section **Project**, sélectionnez **Python Interpreter**.
4. Cliquez sur l'icône + pour ajouter une nouvelle librairie.
5. Recherchez la librairie souhaitée (par exemple, `matplotlib`) dans la barre de recherche.
6. Sélectionnez-la dans la liste et cliquez sur **Install Package**.
7. Attendre que l’installation se termine (un message apparaîtra en bas à gauche).
8. Fermer la fenêtre — la librairie est maintenant prête à être utilisée dans votre code.

Cela installera la librairie dans l'interpréteur Python de votre projet.

---

## Vérifier l'installation

1. Ouvrir un fichier `.py` ou un notebook.
2. Importer la librairie, par exemple :

```python
import matplotlib
```

3. Aucune erreur rouge ne devrait s'afficher. Sinon, redémarrer PyCharm ou revérifier l'installation.

---

:::tip Installer via terminal

On peut aussi installer une librairie en ligne de commande, dans le terminal intégré :

```bash
pip install nom-de-la-librairie
```

Exemple :

```bash
pip install matplotlib
```

:::

---

## Utiliser la librairie dans un fichier Python

```python
import matplotlib.pyplot as plt

plt.plot(range(-10, 11), [x**2 for x in range(-10, 11)])
plt.show()
```

:::tip Versions de librairies

On peut spécifier une version précise lors de l’installation :

```bash
pip install pandas==1.5.3
```

Ou mettre à jour une librairie existante :

```bash
pip install --upgrade pandas
```

:::
